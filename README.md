# Timwork

## 실행 방법

```bash
npm install
npm run dev
```

## 기술 스택

- **React + TypeScript** — UI 컴포넌트
- **Vite** — 빌드 도구
- **Zustand** — 전역 상태 관리
- **react-zoom-pan-pinch** — 도면 줌/패닝 인터랙션
- **Tailwind CSS** — CSS

## 구현 기능

- 전체 배치도 위 건물 폴리곤 클릭으로 도면 선택 및 사이드바 자동 확장
- 전체 배치도 기반 줌/패닝 뷰어
- 확대/축소 초기화 버튼 (화면 맞춤 스케일로 리셋)
- 사이드바 도면 트리 (전체 배치도 → 101동 / 주민공동시설 / 주차장 → 공종 선택)
- 공종 칩 버튼 선택 (공종별 고유 색상 코딩 — 건축=파랑, 구조=주황, 소방=빨강 등)
- 리비전 타임라인 (리비전 존재 시, 하단 바에서 클릭으로 전환)
- 리비전 상세 정보 표시 (발행일, 설명, 변경 내역)
- 상단 컨텍스트 바 (선택된 도면 브레드크럼 표시)
- Single 뷰: 특정 공종/리비전 이미지 단독 표시, 재선택 시(공종 비활성화) 전체 배치도 표시

- Compare 뷰: 최대 4개 레이어 동시 표시 + 투명도 슬라이더 + 오버레이(겹치기 모드), 나란히 모드(최대 2개의 레이어 표시)
- Compare 슬롯 4색 배색 시스템 (파랑/주황/초록/보라, 패널-버튼 동기화)
- Compare 모드 정밀 오버레이 정렬 (computeRelativeTransformCSS 기반 좌표 정렬)
- Compare 선택 규칙 (빌딩 간 차단, relativeTo 좌표계 불일치 차단)
- Compare 리비전 비교 모드 (동일 공종 여러 리비전 동시 오버레이, 자동 전환 감지)

- 구조 공종 Region A/B 분할 처리
- 리비전별 독립 imageTransform 처리 (주민공동시설 건축)
